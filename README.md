# 📚 Bài tập: Cấu trúc dữ liệu và Giải thuật (Tuần 6)


## 📝 Mô tả bài toán
Ứng dụng **Danh sách liên kết đơn (Single Linked List)** để quản lý tệp tin. Các chức năng cốt lõi:
1. Thêm file và tự động sắp xếp theo thứ tự thời gian tăng dần.
2. Tính toán tổng kích thước thư mục.
3. Thuật toán tự động tìm và xóa file có kích thước nhỏ nhất để sao lưu vừa vặn vào ổ USB giới hạn dung lượng (32GB).

## 🚀 Hướng dẫn biên dịch và chạy
Chương trình được viết bằng `C++`. Để chạy chương trình, thực hiện các lệnh sau trong Terminal:

g++ assignmentw6.cpp -o main
