# Bài Tập Tuần 12 - Sắp Xếp Vun Đống (Heap Sort)

**Đề bài:** Cài đặt giải thuật sắp xếp vun đống cho các cây ví dụ trong slide và trên bảng, in ra từng trạng thái lưu trữ cho mỗi bước vun đống và sắp xếp.

## Dữ liệu đầu vào
Chương trình chạy thử nghiệm trên 2 tập dữ liệu:
* Cây ví dụ trong slide: `[4, 1, 3, 2, 16, 9, 10, 14, 8, 7]`
* Cây ví dụ trên bảng: `[203, 106, 78, 84, 59, 15, 32, 16, 10, 17, 38, 11]`

## Hướng dẫn
Biên dịch file `HeapSort.cpp` và chạy để xem log chi tiết từng bước. Kết quả chạy đã được đính kèm trong file ảnh `kq_tuan12.png`.
