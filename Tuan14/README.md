# Bài Tập Tuần 14 - Cây Cân Bằng AVL

Cài đặt thành công cấu trúc cây tự cân bằng AVL bằng ngôn ngữ C++ với chuỗi phần tử đầu vào đặc thù.

## 1. Bộ dữ liệu thử nghiệm
Dãy số nạp vào cây: `32, 51, 27, 83, 96, 11, 45, 75, 66`

## 2. Phết kết quả duyệt cây
* **Duyệt trước (Pre-order):** Hiển thị rõ cấu trúc phân cấp đỉnh và nhánh của cây sau khi trải qua các đợt tự động xoay cân bằng.
* **Duyệt giữa (In-order):** Các phần tử được xuất ra theo đúng thứ tự tăng dần chuẩn mực.
